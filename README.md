# MINI SQL BUILDER

An ASP.NET Core lightweight sql builder for dapper

## Scope Context

The lib is:

* ✓ type-safe SQL construction
* ✓ Columns Mapping
* ✓ Expressions
* ✓ Dapper Integration

The lib isn't:

* ✗ ORM
* ✗ Change Tracking
* ✗ Unit Of Work
* ✗ Lazy Loading
* ✗ Migrations
* ✗ Entities context
